What?

Pagify is a simple jQuery plugin that simply replaces the contents of a div with content from other html files.

Why?

Pagify does nothing that couldn't be done before with good 'ol jQuery.
It does however, provide the following advantages:
  • Easy: Pagify can be setup with only a few lines of code.
  • Clean: Seperate your HTML files.
  • DRY: Options are used to allow for customization and need to only be changed in one place.
  • Configurable: Pagify uses CoC where it makes sense and leaves the rest up to you.
I created Pagify because I noticed that my code for single page sites was getting ugly and I was repeating myself alot.
